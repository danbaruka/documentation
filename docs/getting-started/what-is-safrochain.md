Safrochain empowers communities through **fast, affordable, and accessible
blockchain infrastructure** tailored for real-world use cases. It is a
**Cosmos SDK Layer-1** built for mobile-first users, everyday payments, and
real utility, connecting local African economies to global assets with
sovereign block space, predictable fees, and IBC rails to the wider Cosmos
ecosystem.

## Why Safrochain

- **Everyday payments** with sub-cent effective costs and fast finality.
- **Community savings** (tontines, rotating pools) with transparent,
  programmable rules via CosmWasm.
- **Cross-border settlement** that is verifiable, auditable, and
  interoperable through **IBC**.
- **Predictable fees** so wallets and dApps can quote a real price to the
  user before they sign.

## Technology stack (high level)

- **Cosmos SDK + CometBFT**: proof-of-stake consensus and fast finality.
- **IBC**: cross-chain token transfers and composability with the wider
  Cosmos ecosystem.
- **CosmWasm**: smart contracts for on-chain applications.

## The SAF token

SAF is the network's native token used for:

- **Transaction fees**
- **Staking and validator security**
- **On-chain governance**
