Production send flow for remittance and wallet apps.

```mermaid
sequenceDiagram
  participant User
  participant App
  participant Wallet
  participant SafHandle
  participant Chain

  User->>App: Enter @alice + amount
  App->>SafHandle: resolve handle
  SafHandle-->>App: addr_safro
  App->>Chain: simulate gas
  App->>User: Confirm screen
  User->>Wallet: Approve sign
  Wallet->>Chain: broadcast MsgSend
  Chain-->>App: tx hash
```
